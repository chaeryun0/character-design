## 📌 개요

HTML과 CSS만을 활용하여 나만의 움직이는 캐릭터를 그리는 디자인 대회 참여

<br>

## 🐶 캐릭터 소개

배고파서 애교 부리는 강아지에요. 밥을 먹으면 신이 나서 날아다녀요!

      <li>배가 불렀을 때 치킨을 타고 날아다니는 모습을 표현하기 위해 transform: rotateY() 속성 추가</li>
      <li>box-shadow 속성을 추가하여 입체감 표현</li>
